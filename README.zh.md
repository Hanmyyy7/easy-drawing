## ✨ Descriptions

一般情况下，即使你很好的掌握了canvas和svg，你在绘图过程中仍然会面临很多的挑战。其中最大的挑在于如何轻松地得到坐标，这其中包括各种形状的顶点坐标，以及任何你想知道的相对于某个参照物的坐标。
所以如果可以用canvas去临摹一幅图就太轻松了,基于这样的想法，我选择了使用经纬度在canvas中绘图。
